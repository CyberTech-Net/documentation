Игра:

 - описание
 - правила проведения турниров:
	  количество участников в игре (нужно для формирования команд)

 - история турниров 
 

Турниры:
 - список сущностей типа турнир


Турнир:			(ближайшие турниры фильтр по игре)
 - игра
 - место проведения (время / дата )
 - список команд в турнире
 - призовой фонд
 - рейтинг турнира ( местный , региональный , государственный) 
 - ограничение числа игроков в команде (в зависимости от игры)?
 - обсуждение игры
 - ставки ?
 

Команды
 - список сущностей типа команда  (рейтинг команд) фильтр по игре

Команда	 			(одна команда может участвовать в разных турнирах одного типа игры)
 - Список участников (могут быть пользователи)
 - рейтинг команды:
		- количество проведенных турниров (агрегация по периодам)
		- количество заработанного золота
		- продолжительности жизни команды 
		- количество выигрышей / проигрышей 
		- количество соревнований разного уровня ( региональн / местный/....)

 - Список участий в турнире занятое место / полученный приз  /счет
		

		 
Участник команды:
	- количество проведенных соревнований
	- количество золота
	- список соревнований / участий в них


Чат:
	- между пользователями в привязке к турниру
	- личная перепись



Администратор:
	 - создает новости по игре 
	 - добавляет игры
	 - добавляет турниры ( можно автоматически )


 Пользователь:
	 (авторизированный)
		- может вступать в обсуждения турнира
		- может написать админу
		- может вступить в команду
		- может создать команду
